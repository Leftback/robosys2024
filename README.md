# statistics

![test](https://github.com/Leftback/robosys2024/actions/workflows/test.yml/badge.svg)

## 概要

標準入力から読み込んだ数列に対して、平均値、最頻値、中央値、最大値、最小値、範囲、四分位数、四分位偏差、分散、および標準偏差を計算して出力します。
